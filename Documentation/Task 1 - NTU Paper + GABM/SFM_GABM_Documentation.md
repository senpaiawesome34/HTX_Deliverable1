# SFM + GABM Documentation

## Purpose
Support supervisor follow-up on model spectrum and clarify where SFM + behavioral hybrids sit in evacuation research.

## Key model families
1. Physics-based continuous models (Social Force Model)
2. Cellular automata / floor-field models
3. Rule-based ABM models
4. Behavior-augmented SFM hybrids
5. Generative-hybrid ABM (GABM-like)

## Short takeaways
- SFM is strong for mechanistic movement and interaction effects.
- CA models are efficient for large-scale scenario scans.
- Hybrid approaches are increasingly common for behavior realism.
- GABM-style methods add rich behavior assumptions quickly, but validation burden is high.

## Practical recommendation
Use SFM baseline + hybrid behavior augmentation for research prototyping, but always include explicit uncertainty and external validation plans before policy-level claims.
